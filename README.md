# CodeArena
## 中文名：码斗场
> 如同修罗战场，在此与算法、Bug日夜鏖战，锤炼编程技艺

### 仓库介绍
这是我的专属编程练习竞技场。
存放算法刷题、语言练手、小Demo、踩坑测试、学习实验类代码，
所有与逻辑、语法、边界问题的对峙，皆在此发生。
每一段代码都是一次对决，每一个Bug都是对手。

### 仓库内容分类
```
├── algorithm/      各类算法刷题、数据结构练习
├── language-demo/  Java语言基础语法练习
├── mini-project/   小型练手项目、功能Demo
├── bug-record/     踩坑记录、问题复现代码
├── experiment/     新技术、框架测试实验
└── notes/          配套学习笔记、解题思路
```

### 寄语
以代码为刃，以逻辑为甲，
于这片竞技场，不断突破自身编程边界。

### English Brief
Personal coding training arena.
